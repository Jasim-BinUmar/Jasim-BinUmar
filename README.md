### 👗 Compare Dress
**🌐 [Visit comparedress.com →](https://comparedress.com)**

**Compare Dress** is a fashion price-comparison platform that aggregates products from multiple brands, allowing users to compare prices and availability in one place. I worked on both frontend and backend, building automated scraping pipelines with scheduled cron jobs. Scrapers send success/failure email alerts via SMTP.
